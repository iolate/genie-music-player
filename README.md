# 지니 플레이어 for OS X

## 왜?

* 웹 팝업 플레이어의 한계
  * 디스플레이에 지저분하게 떠다니는 플레이어 팝업
  * 미디어 컨트롤 키 사용 불가
  * 귀찮은 접속 방식
* 지니뮤직의 OS X 플레이어 미지원

<img src="https://user-images.githubusercontent.com/24643489/38609496-305a0f58-3db9-11e8-9ae0-25f9c640e333.png" width="400">

## 주요 기능

* 메뉴 바 트레이 아이콘
  * 재생 정보 제공
* 자동 로그인
